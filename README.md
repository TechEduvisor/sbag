📘 S B A G & Co. LLP – Website README
🏢 Overview.

This project is the official website for S B A G & Co. LLP, a professional firm specializing in:

Bookkeeping

Accounting

Audit Support

US GAAP Compliance  

Payroll Outsourcing

Financial Reporting

Virtual Accounting Team & CPA Support

The website is built for high performance, SEO optimization, cross-platform branding, and fast rendering using modern frontend tooling.

🛠️ Tech Stack
Layer	Technology
Build Tool	Vite
Frontend Framework	React + TypeScript
Styling	TailwindCSS / Custom Components
Icons	Lucide React
Hosting	Any static host (Vercel / Netlify recommended)
